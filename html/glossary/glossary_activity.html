Glossary: Activity by Day

  • # Events
    • The total number of events per day. These incluse forum events, video events, problem events, as well as other events such as Navigation events. Consequently, this # may not add up when summing forum, problem and video events.
  • # Forum Events
    • The number of forum events per day. The edX tracking log data is used for this course.
  • Video Events
    • The number of video events per day. The edX tracking log data is used for this course.
  • Problem Check Events
    • The number of problem check events per day. The edX tracking log data is used for this course.
