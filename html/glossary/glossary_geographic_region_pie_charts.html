Glossary: Geographic Region Pie Charts

Geo-location information is obtained using a user's modal ip address. Note that edX profile location is not used. Additionally, not all ip addresses can be mapped to a location, but most are captured using multiple geo-location databases. Using IP addresses instead of open text fields is more consistent and readily available for a majority of the users that interact with the course. Users who just registered and did not interact with the course (viewers) are not counted in this data.
  • # registered
    • Number of participants who are registered by geographic region.
  • # explored
    • Number of participants who explored by geographic region.
  • # viewed
    • Number of participants who viewed by geographic region.
  • # certified
    • Number of participants who are certified by geographic region.
  • % certified (of registered)
    • Percentage of registrants who are certified.
  • % certified (of viewed)
    • Percentage of learners who viewed the course and certified.
  • avg hours spent on course (registrants)
    • Average number of hours registrants spent in this course.
  • avg hours spent on course (certified)
    • Average number of hours certified learners spent in this course.
  • # verified id
    • Number of participants who got verified ID by geographic region.
  • # verified id certified
    • Number of participants who got verified ID and earned certification by geographic region.
  • % certified (of verified)
    • Percentage of certified users of those received verified IDs.
