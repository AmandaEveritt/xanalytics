Glossary: Grade histograms by chapter for course

  • chapter_mid
    • module-id of chapter within which this module element exists
  • chapter_name
    • human-readable name for this module element of the course
  • due_date
    • due date for this problem
  • course_axis_index
    • sequence index giving order in which this problem appears within the course
  • chapter_max_grade
    • Maximum grade achievable within chapter of the course
  • n_total
    • Total # of problems within the chapter
