Glossary: Content by Chapter

  • Time Index
    • The order of which this content is added to the course.
  • Chapter Name
    • The title of the chapter.
  • Start date
    • The content chapter release date.
  • User Views
    • The number of users viewed this chapter (i.e., clicked on the chapter title)
  • Module Category
    • The category of content (e.g., video, html, problem)
  • Module Name
    • The title of the content.
  • Submissions
    • Total number of assignment submissions.
  • AVG Grade
    • Average student grade for this problem.
  • MAX Grade
    • Maximum grade attained by students for this problem.
  • AVG Attempts
    • Average number of attempts at solving problems in this chapter across all problems.
  • MAX Attempts
    • Maximum attempts at solving problems in this chapter across all problems.
  • Start Date
    • The content chapter release date.
  • User Views
    • The number of users who viewed this chapter material.
